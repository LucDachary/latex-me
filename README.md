Latex Me
===
A command line to bootstrap Latex documents right in the shell.

# Usage
```shell
$ latexme article myarticle.tex
```

# Requirements
Tcl/Expect is required. On Archlinux simply install `expect`:
```shell
sudo pacman -S expect
```
